# Heavy Duty Training Log

This log keeps training minimal, measurable, and recovery-aware.

## Template

| Day | Focus | Work |
| --- | --- | --- |
| Pull | back / biceps | one top set per movement |
| Push | chest / shoulders / triceps | controlled progression |
| Legs | quads / hamstrings / calves | high effort, low volume |

```text
movement: incline press
warmups: 2
top_set: 6-9 reps
progression: add load after upper bound
```

The principle is not novelty. It is precise stimulus, enough recovery, and honest logging.
