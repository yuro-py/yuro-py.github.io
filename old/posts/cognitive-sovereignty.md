# Cognitive Sovereignty and AI

AI should amplify thinking without replacing judgment. The central concern is not whether tools are powerful, but whether the human operator remains awake inside the loop.

## Working Pillars

| Pillar | Practice |
| --- | --- |
| Cognitive sovereignty | keep independent models of the problem |
| Craft integrity | inspect the work, not just the output |
| Moral agency | own decisions made with tools |
| Discernment | separate speed from correctness |
| Epistemic independence | verify claims against sources |
| Relational presence | do not outsource human attention |

The useful rule is simple: use automation for leverage, not abdication.
